#簡易註冊/登入作品

為學習後端而寫的小作品，可註冊及登入，且已部署至heroku，可至demo網址測試。

![首頁](https://i.imgur.com/lWLB3Ux.png "首頁")

demo網址：[簡易註冊/登入](https://infinite-plateau-31058.herokuapp.com/)

## 頁面架構
* 前台：
   + 首頁
   + 使用者頁面(=登入後頁面)
   + 登入頁面
   + 註冊頁面
   
## 使用技術 + 套件
* node.js
* express
* firebase
* pug
* Bootstrap4
* express-session
* express-validator
* connect-flash
* dotenv
